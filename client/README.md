# BetixPro Client

This is the frontend application for the BetixPro  Betting Management System.

## 🚀 Technologies
- **React 19**
- **Vite**
- **Tailwind CSS v4**
- **TanStack Query & Router**
- **Radix UI**

## 🛠️ Setup
Please refer to the [Main README](../README.md) for full project setup instructions.

### Scripts
- `pnpm dev`: Start the development server.
- `pnpm build`: Build the application for production.
- `pnpm lint`: Run ESLint to check for code quality issues.
- `pnpm preview`: Preview the production build locally.
