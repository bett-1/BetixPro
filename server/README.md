# BetixPro Server

This is the backend API and database management layer for the BetixPro  Betting Management System.

## 🚀 Technologies
- **Node.js & Express**
- **Prisma ORM**
- **PostgreSQL**
- **Socket.io**
- **JWT Authentication**

## 🛠️ Setup
Please refer to the [Main README](../README.md) for full project setup instructions.

### Scripts
- `pnpm dev`: Start the development server with nodemon.
- `pnpm build`: Compile TypeScript to JavaScript.
- `pnpm start`: Start the production server.
- `pnpm prisma:migrate`: Run database migrations.
- `pnpm prisma:seed`: Seed the database with initial/test data.
- `pnpm db:deploy`: Deploy database migrations and seed in production.
